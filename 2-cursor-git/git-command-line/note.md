# Git 常用命令笔记整理

## 1. 初始化与克隆项目

- `git init`  
  初始化一个本地 Git 仓库。

- `git clone <远程仓库地址>`  
  （例如：https/ssh url）克隆远程项目到本地。

---

## 2. 查看与添加文件状态

- `git status`  
  查看当前仓库状态（有哪些文件修改/未追踪等）。

- `git add <文件或目录>`  
  添加文件到暂存区，准备提交。

---

## 3. 提交与日志

- `git commit -m "说明"`  
  提交暂存区内容到仓库（写上提交说明）。

- `git log`  
  查看提交历史日志。

---

## 4. 分支管理

- `git branch`  
  列出所有分支。

- `git branch <分支名>`  
  创建新分支。

- `git checkout <分支名>`  
  切换到指定分支。

- `git checkout -b <新分支名>`  
  新建并切换到该分支。

- `git switch <分支名>`  
  切换分支（新版推荐）。

- `git switch -c <新分支名>`  
  创建并切换新分支（新版推荐）。

---

## 5. 远程操作

- `git push`  
  推送更改到远程仓库。

- `git pull`  
  获取远程仓库最新内容并合并到本地。

---

## 6. 合并与回退

- `git merge <分支名>`  
  合并其他分支到当前分支。

- `git reset <提交ID>`  
  回退到指定的提交记录（危险操作，慎用）。
